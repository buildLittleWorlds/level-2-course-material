# Story Emotion Arc — 3-Class Sentiment

Paste a multi-paragraph story and watch how positive, negative, and neutral sentiment shifts across the narrative. The simplest emotional lens.

Built with React + TypeScript + Tailwind CSS. Uses the Hugging Face Inference API with `cardiffnlp/twitter-roberta-base-sentiment-latest`.
