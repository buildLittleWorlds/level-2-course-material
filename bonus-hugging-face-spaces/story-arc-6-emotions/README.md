# Story Emotion Arc — 6 Emotions

Paste a multi-paragraph story and watch how sadness, joy, love, anger, fear, and surprise rise and fall. This model includes love — absent from Ekman's classic set.

Built with React + TypeScript + Tailwind CSS. Uses the Hugging Face Inference API with `bhadresh-savani/distilbert-base-uncased-emotion`.
