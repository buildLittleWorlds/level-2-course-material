# Story Emotion Arc — 7 Ekman Emotions

Paste a multi-paragraph story and watch how anger, disgust, fear, joy, neutral, sadness, and surprise rise and fall across the narrative.

Built with React + TypeScript + Tailwind CSS. Uses the Hugging Face Inference API with `j-hartmann/emotion-english-distilroberta-base`.
