# Story Emotion Arc — 28 GoEmotions

Paste a multi-paragraph story and watch how 28 fine-grained emotions (admiration, curiosity, grief, and more) shift across the narrative.

Built with React + TypeScript + Tailwind CSS. Uses the Hugging Face Inference API with `SamLowe/roberta-base-go_emotions`.
