# Session 1: From Rules to Models
*"Can a Computer Tell How You Feel?"*

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/buildLittleWorlds/level-2-course-material/blob/main/session-01/notebook.ipynb)

## What This Session Covers

Students explore Hugging Face Spaces that perform sentiment analysis, learn the INPUT → MODEL → OUTPUT pattern, and start curating their first Collection. The session introduces the Mood Meter Space and five bonus Spaces as exploration targets. The Big Question: can a machine actually feel, or is it just matching patterns?
