The Code: Imports

import gradio as gr
from transformers import pipeline
gradio
Builds the web interface
transformers
Loads the AI model

Two imports. That's it.

The Code: Load the Model

analyzer = pipeline(
    "sentiment-analysis",
    model="distilbert-base-uncased-finetuned-sst-2-english"
)

One line loads the AI model.
pipeline handles all the complicated stuff.
"sentiment-analysis" is the task — reading feelings.
The model name tells it which brain to download.

The Code: The Function

def check_mood(text):
    if not text or not text.strip():
        return "Paste some text above first!"

    result = analyzer(text)[0]
    label = result["label"]
    score = result["score"]

    if label == "POSITIVE":
        emoji = "😊" if score > 0.9 else "🙂"
    else:
        emoji = "😢" if score > 0.9 else "😐"

    return f"{emoji} {label}\n\nConfidence: {score:.0%}"

Text goes in. The model reads it. A mood comes out.

The Code: The Web Interface

demo = gr.Interface(
    fn=check_mood,
    inputs=gr.Textbox(lines=8,
        placeholder="Paste any text here..."),
    outputs=gr.Textbox(label="Mood Reading"),
    title="Mood Meter",
    description="Paste any text and this AI will "
                "tell you whether it feels POSITIVE "
                "or NEGATIVE..."
)
demo.launch()

fn = your function • inputs = what the user types • outputs = what they see

What's "Sentiment Analysis"?

The model distilbert-base-uncased-finetuned-sst-2-english was trained on movie reviews.

Someone labeled thousands of movie reviews as POSITIVE or NEGATIVE. The model learned the patterns of each.

Now it reads any text through that lens —
even diary entries, song lyrics, and text messages.
