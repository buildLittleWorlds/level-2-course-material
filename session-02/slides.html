The Punchline

The training data determines what the model can see.

The sentiment model was trained on movie reviews.
So it thinks everything is either a good movie or a bad movie.
The emotion model was trained on tweets.
It understands internet language — but struggles with formal text.
The zero-shot model was trained on general language.
It can handle anything you throw at it — but less precisely.

The Code: Emotion Detector

import gradio as gr
from transformers import pipeline
import re

# Load the emotion model (trained on tweets)
classifier = pipeline(
    "text-classification",
    model="j-hartmann/emotion-english-distilroberta-base"
)

One line changes the entire behavior.
Different pipeline type, different model, different output.

The Code: Analyze Each Sentence

def analyze_text(text):
    if not text or not text.strip():
        return "Type or paste some text above first!"

    sentences = [s.strip() for s in
        re.split(r'(?<=[.!?])\s+(?=[A-Z"])', text)
        if len(s.strip()) > 10]

    if len(sentences) == 0:
        return "I need at least one full sentence."

    output_lines = []
    for sentence in sentences:
        result = classifier(sentence)[0]
        label = result["label"]
        score = result["score"]
        output_lines.append(
            f"{label.upper()} ({score:.0%}): {sentence}"
        )

    return "\n\n".join(output_lines)

The Code: Interface

demo = gr.Interface(
    fn=analyze_text,
    inputs=gr.Textbox(lines=8,
        placeholder="Type or paste some text here..."),
    outputs=gr.Textbox(label="Emotion Analysis", lines=12),
    title="Emotion Detector",
    description="Paste any text and see what emotion "
                "the AI detects in each sentence.",
)
demo.launch()

Name the Concept

TRAINING DATA AND REPRESENTATION

same INPUT + different TRAINING DATA → different OUTPUT

"Training data is the textbook the model studied from.
It can only know what it's been shown."

Representation = the categories the data uses.
Positive/negative is one representation. Seven emotions is another.

Vocabulary

Training data — the examples a model learned from
Representation — the categories or labels the training data uses
Sentiment — positive or negative feeling
Model card — documentation that describes what a model does and how it was trained
Fine-tuned — a model further trained on specific data for a specific task
