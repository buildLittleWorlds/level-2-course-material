## What This Session Covers

Students feed sarcastic and adversarial inputs to a sentiment model and watch it fail. They build a `clean_text()` preprocessing function to handle tricky inputs, then test before-and-after results to see the difference data cleaning makes. The concept: Data Cleaning and Feature Engineering. The Big Question: if the model can't understand sarcasm, whose fault is that?
