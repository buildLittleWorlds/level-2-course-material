The Compute Problem

A serious LLM is dozens of gigabytes. Free Hugging Face CPU is small.

You can't fit Gemini, GPT-4, or Claude inside your Space.

But you don't have to. You can call one over the network.

The Deeper Question

When your Space connects to a hosted model, your tool depends on a contract — what you send, what you expect back.

Contracts are honored most of the time.

When they break — the model returns prose instead of JSON, or hallucinates a result, or hits a rate limit — your downstream code has nowhere to hide.

Tonight you'll see what that looks like.

The Code: Configure with the Secret

api_key = os.environ.get("GEMINI_API_KEY")
if not api_key:
    raise RuntimeError(
        "GEMINI_API_KEY not set. Add it as a Secret in your "
        "Space's Settings tab."
    )

genai.configure(api_key=api_key)
model = genai.GenerativeModel("gemini-1.5-flash")

The key lives in HF Secrets, not in your code.
Public source code ≠ public credentials.

The Code: The Prompt Contract

PROMPT_TEMPLATE = """You are a media analyst. Rate the political
bias of this news headline on a scale from -5 (strong left-leaning
framing) to +5 (strong right-leaning framing), where 0 is neutral.
Return ONLY a JSON object with these keys:
  - "score": integer from -5 to 5
  - "label": one of "left", "left-leaning", "neutral",
             "right-leaning", "right"
  - "rationale": one sentence

Headline: {headline}

Return only the JSON. No prose, no markdown fences."""

The prompt is the contract.
When the model honors it, your code works. When it doesn't, your code breaks.

The Code: Call & Parse

def analyze(headline):
    if not headline.strip():
        return "", "", "", "", "Status: waiting for input"

    prompt = PROMPT_TEMPLATE.format(headline=headline.strip())
    raw = model.generate_content(prompt).text

    try:
        data = json.loads(raw)
    except json.JSONDecodeError:
        return ("", "", "", raw,
                "Status: PARSE FAILED — the contract broke")

    return (str(data["score"]), data["label"],
            data["rationale"], raw, "Status: clean JSON received")

Output of Model 1 (Gemini's text) becomes input of Model 2 (your parser).

API Connections in the Real World

💬 ChatGPT plugins
Your prompt → tool API call → structured response → final answer
If the tool returns garbage, the answer confidently repeats it.
📱 Apps using OpenAI / Claude / Gemini
Almost every "AI feature" you see is an API call to a hosted model
The app's job is mostly prompt construction and response parsing.
🏥 Medical AI tools
Symptom input → LLM API → JSON of conditions → UI
When the JSON contract breaks, downstream displays fail or mislead.

Vocabulary

API — a clearly-defined channel for one piece of code to ask another piece of code to do something
Secret — a credential (like an API key) stored separately from your code so it isn't visible in public source
JSON contract — the agreed shape of a structured response; when it breaks, the parser fails
Rate limit — the cap on how many requests you can send per minute or day
Hallucination as upstream error — a confident-sounding wrong answer your code can't detect
Garbage in, garbage out — if any link in the chain is wrong, everything after it is wrong
Cold start — the delay when a Space wakes up and reconnects to its API
