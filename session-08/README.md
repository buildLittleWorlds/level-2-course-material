## What This Session Covers

Students chain BLIP image captioning with sentiment analysis — upload an image, generate a caption, then analyze the caption's sentiment. They learn how multi-model systems work and discover error cascades: when the first model gets it wrong, everything downstream goes wrong too. The concept: Multi-Model Systems and Error Cascades. The Big Question: what happens when AI talks to AI?
