# Parallel Example: Perspective Probe

This packet models how a visual-perspective Hugging Face Space idea could become a research journal and paper-lite draft. It is adjacent to camera-angle and scene-description work, but it does not describe the student's actual project.

## Core idea

The imagined Space is a **Perspective Probe**. A user gives a scene prompt and asks text and image tools to describe or render the scene from different viewpoints. The project asks whether the tools actually change spatial relationships or simply add viewpoint words.

## Research question

When language and image tools are asked to change perspective, do they actually change the scene in a spatial way, or do they mostly swap in perspective-coded words and visuals?

## Space idea

Inputs:

- Scene prompt
- Viewpoint: close-up, wide shot, bird's-eye view, low angle, over-the-shoulder
- Mode: text description, image prompt, or comparison table

Outputs:

- Generated scene text
- Perspective-specific image prompt or image output
- Checklist for spatial consistency
- Reflection prompt asking what changed besides viewpoint vocabulary

## What this packet demonstrates

- How to turn "camera angle" into a researchable spatial-reasoning question
- How to compare text and image outputs as two forms of evidence
- How to connect classroom testing to research on novel view synthesis and camera control
- How to make a claim about shallow versus genuine perspective change
