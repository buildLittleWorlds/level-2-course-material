# Session 10: Build Your Own from Scratch

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/buildLittleWorlds/level-2-course-material/blob/main/session-10/notebook.ipynb)

## What This Session Covers

Students choose a model, design a task, and build a complete Hugging Face Space independently for the first time. They work from starter templates and make every decision themselves — what the tool does, who it is for, and how it should look. The concept: Supervised Learning and Task Design. The Big Question: now that you know how these pieces work, what do you want to build?
