## What This Session Covers

Students redesign a sentiment analysis demo into a tool for a real audience — a restaurant owner tracking reviews, a teacher reading student feedback, or a therapist monitoring journal entries. They learn how prompt engineering and interface design shape what AI can actually do for people. The concept: Prompt Engineering and Human-AI Interaction. The Big Question: what turns a demo into something someone would actually use?
